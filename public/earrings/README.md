# Earring Images

Please add your earring style images here in WebP format with transparent backgrounds.

## Required Files:
- `gold-stud.webp` - Classic gold ball stud earring
- `pearl-drop.webp` - Pearl drop earring with gold hook
- `silver-hoop.webp` - Medium-sized silver hoop earring
- `crystal-chandelier.webp` - Glamorous cascading crystal chandelier earring
- `rose-gold-hoop.webp` - Small rose gold hoop earring
- `diamond-stud.webp` - Diamond solitaire stud earring

## Image Requirements:
- Format: WebP with transparency
- Orientation: Single earring (not a pair)
- Background: Transparent
- Size: Recommended 500-1000px width
- Quality: High resolution for clear rendering

## Tips:
- Images should show the earring as it would hang/sit on the ear
- For studs: show from the front
- For drops/chandeliers: show hanging vertically
- For hoops: show the circular shape clearly
